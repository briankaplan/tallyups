## GAME CONCEPT

### One-Line Pitch
*"Sort colorful objects into satisfying stacks, merge for rewards, feel the ASMR satisfaction"*

### Core Gameplay Loop
```
1. Colorful objects (balls/gems/shapes) appear on screen
2. Player swipes to sort into matching colored containers
3. Containers fill → Satisfying merge animation + sound + haptic
4. Variable rewards drop (coins, gems, rare items)
5. Difficulty scales (more colors, faster speed, obstacles)
6. Session ends with hook: "Next level unlocks NEW MECHANIC!"
```

### Unique Differentiators
1. **Haptic Symphony** - Each action has unique haptic feedback pattern
2. **ASMR Audio Design** - TikTok-optimized satisfying sounds
3. **No Forced Ads** - Royal Match model (player-friendly = higher LTV)
4. **Dynamic Difficulty** - AI maintains flow state (not too hard/easy)
5. **3-Minute Sessions** - Perfect for social media sharing

## PSYCHOLOGICAL HOOKS (Research-Backed)

### 1. Variable Ratio Reinforcement (Skinner)
```
Every successful sort:
  70% → Standard coins (5-15)
  20% → Bonus coins (25-50) + special sound
  8%  → Rare gem drop + particle explosion
  2%  → JACKPOT (100+ coins) + screen shake + celebration
```
**Why**: Unpredictable rewards create strongest behavioral persistence (slot machine psychology without gambling)

### 2. The Hook Model (Nir Eyal)
```
TRIGGER → Push notification: "Your streak is waiting!"
ACTION → Open app, tap play (< 3 taps to gameplay)
VARIABLE REWARD → Random bonus, level completion, new unlock
INVESTMENT → Progress saved, streak extended, collection grows
```

### 3. Flow State (Csikszentmihalyi)
- Dynamic difficulty adjustment (AI monitors performance)
- If player fails 3x → slightly easier
- If player wins easily → slightly harder
- Goal: Keep in "flow zone" between anxiety and boredom

### 4. Near-Miss Effect
- Containers fill to 95%... need just one more
- "SO CLOSE!" messages on near-completions
- Visual tease of almost-won rewards
- **Optimal rate**: 30% near-misses (research-backed)

### 5. Streak System (Ethical Design)
```
Day 1:   50 coins
Day 2:   75 coins
Day 3:   100 coins + Random Booster
Day 5:   200 coins + Rare Skin
Day 7:   500 coins + Exclusive Item + Chest
Day 14:  1,000 coins + Epic Item
Day 30:  5,000 coins + Legendary Item + Title

PROTECTION:
- 1 FREE Streak Freeze per week
- Additional freezes: 50 gems each
- 24-hour grace period before streak breaks
```

### 6. Endowed Progress Effect
- New players start at "Level 1 of 50" (not 0)
- Tutorial completion = "25% of first world done!"
- Collection album shows 3 items pre-unlocked
- Progress bars everywhere (always almost complete)

### 7. Zeigarnik Effect (Incomplete Tasks)
- Always show "Next reward: 3 levels away"
- Daily challenge visible but incomplete on home screen
- "New world unlocks in 5 levels" teaser
- Session ends with preview of next mechanic

### 8. Loss Aversion
- Streak protection framing: "Protect your 15-day streak!"
- "Don't lose your progress" on app close
- Energy system recovery messaging

### 9. FOMO (Ethical Implementation)
- Weekend events with COSMETIC-ONLY exclusives
- Seasonal themes (Halloween, Christmas skins)
- Limited-time Sort Pass rewards (30-day seasons)
- "Event ends in 2 days" - genuine scarcity

### 10. Social Proof
- "Join 2.5 million players!"
- "Only 5% of players beat Level 100"
- Friend leaderboards
- Global weekly rankings

- Direct Core Haptics integration
- Guaranteed 60fps on all devices
- Metal backend for GPU acceleration
- Best-in-class iOS integration (Game Center, StoreKit)
- No licensing fees or royalties

### Key Technical Features

#### 1. Haptic Feedback System
```swift
// HapticManager.swift - Core Haptics integration
- sortLight: UIImpactFeedbackGenerator.style.light
- sortMedium: UIImpactFeedbackGenerator.style.medium
- sortHeavy: UIImpactFeedbackGenerator.style.heavy
- mergeComplete: Custom CHHapticPattern (rising intensity)
- jackpot: Complex pattern with multiple transients
- levelComplete: Success notification pattern
```

#### 2. ASMR Audio Design
```
Sound Categories:
- Ambient: Soft background loop (optional)
- Sort sounds: Satisfying "pop" variations
- Merge sounds: Whoosh + chime combination
- Reward sounds: Coin shower, gem sparkle
- UI sounds: Soft taps, swishes
```

#### 3. Physics System
```swift
// SpriteKit physics for satisfying feel
- Gravity: Adjustable per level
- Bounce: Slight bounce on container entry
- Particle systems: Confetti, sparkles, explosions
- Smooth animations: Spring damping for natural feel
```

#### 4. Dynamic Difficulty
```swift
// DifficultyManager.swift
- Track: consecutiveWins, consecutiveLosses, avgCompletionTime
- If consecutiveLosses >= 3: reduce speed 10%, add 5 seconds
- If consecutiveWins >= 5: increase speed 5%, reduce time 3 seconds
- Always stay within "flow zone" bounds
```
